# Folder Visualizer - 简历项目描述（200 字精简版）

## 项目名称
Folder Visualizer（文件夹可视化工具）

## 一句话介绍
一款零依赖的 Windows 原生文件夹结构可视化与高清导出工具

## 项目简介（150 字）
基于 C++17 和 Win32 API 开发的轻量级桌面工具。支持快速扫描文件夹并以树形结构展示，自动识别 20+ 种文件类型并映射自定义图标。可导出为 TXT 或高分辨率 PNG 图片（3200px 宽）。采用 GDI+ 进行高质量渲染，使用 DIB 位图技术优化输出质量。程序静态链接所有库，单文件分发（~3.5MB），无需任何运行时依赖。

## 技术栈（30 字）
C++17 / Win32 API / GDI+ / CMake / MinGW / Windows Resource Compiler

## 核心亮点（50 字）
- 零运行时依赖，单文件开箱即用
- 高性能渲染，支持万级文件扫描
- 资源嵌入技术（图标、版本信息编译到 exe）
- 现代 C++ 实践（智能指针、RAII）

## 差异化优势
相比命令行 tree：图形界面 + 高清图片导出  
相比第三方工具：纯原生开发 + 零依赖 + 体积小巧

## 项目数据
- 代码量：~1500 行 C++
- 模块数：5 个核心模块
- 支持：20+ 文件类型，100+ 扩展名

## 适用场景
项目文档生成、代码仓库展示、文件系统教学
