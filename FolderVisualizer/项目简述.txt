# Folder Visualizer 项目简述

## 项目名称
Folder Visualizer（文件夹可视化工具）

## 项目定位
一款轻量级、高性能的 Windows 文件夹结构可视化与导出工具

## 核心功能
- **快速扫描**：支持万级文件的文件夹结构快速扫描与渲染
- **树形展示**：美观的树形可视化界面，带连接线和文件类型图标
- **智能分类**：自动识别 20+ 种文件类型（代码、图片、音频、视频、文档等）
- **高清导出**：支持导出为 TXT 文本或高分辨率 PNG 图片（3200px 宽，2x 缩放）
- **自定义图标**：支持按文件扩展名映射自定义图标

## 技术栈
- **开发语言**：C++17（智能指针、Lambda、文件系统）
- **GUI 框架**：Win32 API（原生 Windows 界面，零依赖）
- **图形渲染**：GDI+（高质量 2D 绘图，DIB 位图）
- **构建系统**：CMake + MinGW-w64
- **资源管理**：Windows Resource Compiler（嵌入图标和版本信息）

## 技术亮点
1. **零运行时依赖**：静态链接所有库，单文件分发，开箱即用
2. **高性能渲染**：使用内存 DC 和 DIB 位图，支持大尺寸图像导出
3. **资源嵌入**：程序图标和版本信息直接编译到 exe，无需外部资源文件
4. **现代 C++ 实践**：RAII 内存管理、智能指针、函数式编程
5. **可扩展架构**：模块化设计（扫描、渲染、图标管理分离）

## 差异化特色
- **vs 命令行 tree**：提供图形界面和高清图片导出能力
- **vs 资源管理器**：支持文件类型分类统计和自定义图标映射
- **vs 第三方工具**：零依赖、单文件、纯原生开发，体积小巧（~3.5MB）

## 项目数据
- **代码量**：约 1500 行 C++ 代码
- **模块数**：5 个核心模块（GUI、FileTree、FileTypeManager、IconManager、TreeNode）
- **支持格式**：20+ 文件类型分类，100+ 扩展名识别
- **性能指标**：单目录扫描上限 10000 文件，最大递归深度 100 层

## 适用场景
- 项目目录结构文档生成
- 代码仓库结构展示
- 文件系统教学演示
- 批量文件整理前的结构预览

## 运行环境
- Windows 7 及以上版本（32/64 位）
- 无需安装任何运行时库

## 获取方式
编译后单文件 `FolderVisualizer.exe`，可直接分发使用
