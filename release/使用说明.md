# Folder Visualizer 使用说明

## 1. 项目简介
Folder Visualizer 是一款 Windows 桌面工具，用于扫描文件夹并将目录结构可视化展示，支持导出文本和图片结果。

## 2. 核心功能
- 扫描指定目录并生成树形结构
- 按文件类型分类显示（代码、文档、图片、压缩包等）
- 图标映射展示（可通过配置扩展）
- 导出为 TXT 文本
- 导出为 PNG 高清图片

## 3. 运行方式
可执行文件位于：
- `release/FolderVisualizer.exe`

双击运行后：
1. 点击“浏览...”选择目录
2. 点击“开始扫描”
3. 扫描完成后可点击“保存TXT”或“保存图片”导出

## 4. 配置文件
程序默认读取可执行文件同目录下的 `settings.json`：
- `release/settings.json`

常用配置项：
- `folderVisualizer.iconPath`：图标目录（如 `icons/`）
- `folderVisualizer.defaultFolderIcon`：默认文件夹图标
- `folderVisualizer.defaultFileIcon`：默认文件图标
- `folderVisualizer.windowIcon`：窗口图标文件名
- `folderVisualizer.default{ext}Icon`：扩展名图标映射（如 `defaultcppIcon`）

## 5. 构建方式（MinGW + CMake）
本项目已切换为纯 CMake 构建，编译器为 MinGW。

### 推荐方式（Preset）
```bash
cmake --preset mingw-release
cmake --build --preset mingw-release
```

### 直接命令
```bash
cmake -S . -B build_mingw -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release
cmake --build build_mingw
```

## 6. 构建产物
- 可执行文件输出：`release/FolderVisualizer.exe`
- 运行资源自动复制到：`release/`
  - `settings.json`
  - `tree.ico`
  - `icons/`

## 7. 目录说明（当前结构）
```text
FolderVisualizer/
├─ assets/                # 运行资源源目录（icons、settings、ico）
├─ build_mingw/           # MinGW CMake 构建目录
├─ release/               # 发布目录（exe 与运行资源）
├─ CMakeLists.txt
├─ CMakePresets.json
├─ FolderVisualizerGUI.cpp
├─ FileTree.cpp/h
├─ FileTypeManager.cpp/h
├─ IconManager.cpp/h
├─ TreeNode.cpp/h
├─ resource.rc
└─ resource.h
```

## 8. 注意事项
- 若图标不显示，请检查 `release/icons/` 是否存在且文件名与配置一致。
- 扫描超大目录可能需要一定时间。
- 项目已限制为 MinGW 构建；若使用 MSVC 配置会被 CMake 拒绝。
