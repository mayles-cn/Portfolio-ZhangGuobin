# Folder Visualizer - 简历项目描述（精简版）

## 项目名称
Folder Visualizer（文件夹可视化工具）

## 一句话介绍
一个基于 C++17/Win32 API 的 Windows 原生目录结构可视化与导出工具，支持一键扫描、树形展示和高分辨率图片导出。

## 项目简介
该项目面向“目录结构展示和归档”场景，提供本地文件夹的快速扫描、分类显示与导出能力。程序可识别常见文件类型（代码、图片、文档、压缩包、可执行文件等），并通过图标映射提升结构可读性。支持导出 TXT 文本结构和 PNG 高清图片，适用于项目交付文档、代码仓库结构说明、教学演示等场景。

## 技术栈
- C++17
- Win32 API
- GDI+
- CMake
- MinGW-w64

## 个人贡献亮点
- 设计并实现文件树扫描与递归渲染流程，支持大目录稳定展示。
- 实现文件类型识别与扩展名图标映射机制，提升可视化表达能力。
- 实现 TXT/PNG 两种导出链路，支持高分辨率图像输出。
- 重构构建系统为纯 CMake + MinGW，统一产物输出到 `release/`。

## 项目结果
- 完成可独立运行的 Windows 可执行程序。
- 支持配置化图标与资源拷贝流程，发布目录结构清晰。
- 可用于实际项目目录可视化和文档化输出。
